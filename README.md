# HC Encyclopedia

Open-source помощник для игроков Hustle Castle.

## Возможности

- синхронизация базы SageHC
- локальная база SQLite
- несколько игровых профилей
- оптимизация рун
- оптимизация экипировки
- анализ состава команды

## Установка

```bash
pip install -e .
```

## Запуск

```bash
hcenc --help
```
